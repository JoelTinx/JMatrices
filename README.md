# JMatrices: #
Es un programa escrito en java, que realiza operaciones tanto básicas como avanzadas con matrices.

## Operaciones que realiza: ##
- Suma / resta
- Multiplicación
- Determinante
- Inversa
- Adjunta
- Transpuesta
- Traza
- Matriz Escalonada
- Cuadrado mágico

## Otras funcionalidades: ##
- lectura y escritura de matrices ingresadas.
- Propia extensión
